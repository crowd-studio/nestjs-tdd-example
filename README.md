# nestjs-tdd-example

### Init project

Install nest cli
```bash
yarn global add @nestjs/cli
```
Clone demo repo
```bash
git clone https://github.com/crowd-studio/nestjs-tdd-example.git
```

Install dependencies
```bash
yarn
```

### Develop

Create users module
```bash
nest g res users
```
